# 快速启动指南

## 方式一：使用启动脚本（推荐）

双击 `start.bat` 文件，将自动：
1. 检查并安装依赖
2. 启动后端服务器（http://localhost:3001）
3. 启动前端服务器（http://localhost:3000）

## 方式二：手动启动

### 1. 安装依赖
```bash
# 前端
cd apps/web
npm install

# 后端
cd ../server
npm install
```

### 2. 启动服务
```bash
# 后端（新终端窗口）
cd apps/server
npm run dev

# 前端（新终端窗口）
cd apps/web
npm run start
```

## 使用步骤

1. 访问 http://localhost:3000
2. **首次使用会看到黄色警告提示**
3. 点击导航栏右侧的"设置"按钮（或在警告提示中点击"立即配置"）
4. 在设置页面配置 LLM：
   - Base URL: 如 https://api.openai.com/v1
   - API Key: 你的 API Key
   - Model: 如 gpt-4
5. 点击"测试连接"确认配置成功
6. 点击导航栏的"开营"按钮返回首页
7. 输入一句话需求，如：
   - "下周五办一场面向新手的华为云数据库实战营"
   - "下周一开始为期7天的AI大模型入门训练营"
8. 点击"一键开营"，等待生成完成（约30-60秒）
9. 自动跳转到活动详情页，查看和编辑生成的物料

## 界面说明

### 导航栏（顶部）
- **开营**：一键开营主页面
- **活动列表**：查看所有已生成的活动
- **设置**：配置 LLM API

### 首页（开营页）
- 如果未配置 LLM，会显示黄色警告提示
- 输入框：输入一句话需求
- 示例需求：点击可快速填充
- 一键开营按钮：开始生成

### 活动详情页
- **概览**：查看活动基本信息和二维码
- **海报**：预览、编辑、下载 HTML 海报
- **表单**：查看和编辑报名表单字段
- **文案**：查看和编辑每日文案，支持一键复制

## 常见问题

### Q: 前端启动报错？
A: 确保已安装所有依赖：
```bash
cd apps/web
npm install
```

### Q: 后端启动报错？
A: 确保已安装所有依赖：
```bash
cd apps/server
npm install
```

### Q: LLM 连接失败？
A: 检查：
- Base URL 是否正确（注意要包含 /v1）
- API Key 是否有效
- 模型名称是否正确
- 网络是否能访问 API

### Q: 生成失败？
A: 可能原因：
- LLM 配置不正确
- API 限流，稍后重试
- 需求描述不够清晰，尝试更详细的描述

## 技术支持

如遇问题，请查看：
- 后端日志：后端终端窗口
- 前端日志：浏览器控制台（F12）
